DH43
Introduction to Python: Python for Social Science

Canvas Page

Winter 2025

This class is an introduction to data science in the social sciences with an emphasis on the Python programming language. Students with no prior experience in programming and data science will learn about reading, manipulating, and analyzing data. The class also offers students the opportunity to learn about methods used to analyze quantitative data, textual data, and spatial data using real-world data sets.
